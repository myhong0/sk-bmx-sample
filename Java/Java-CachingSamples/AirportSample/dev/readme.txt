Setup a catalog server and container server for testing

1.  Copy the airportGrid folder to <wlp>\usr\servers folder
2.  Start the server by running "server start airportGrid --clean" in the <wlp>/bin folder

Setup a web server for testing (Using JNDI ObjectGrid access configuration)

1.  Copy the airportCloud folder to <wlp>\usr\servers folder
2.  Start the server by running "server start airportCloud --clean" in the <wlp>/bin folder
